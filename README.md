# 🧠 Behavioural Portfolio Optimizer

> A comprehensive behavioural finance platform built for ZeTheta Algorithms

## 🚀 Quick Start

### Backend
\`\`\`bash
cd backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
uvicorn main:app --reload
\`\`\`

### Frontend
\`\`\`bash
cd frontend
npm install
npm run dev
\`\`\`

### Tests
\`\`\`bash
cd backend
pytest tests/ -v
\`\`\`
